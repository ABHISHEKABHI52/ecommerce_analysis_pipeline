================================================================================
                        HOW TO PUSH TO GITHUB
================================================================================

Method 1: Using Git Commands (Recommended)
────────────────────────────────────────────────────────────────────────────────

Step 1: Open Command Prompt or Terminal
    - Press: Windows Key + R
    - Type: cmd
    - Press: Enter

Step 2: Navigate to Project
    cd C:\Users\abhis\TT_Project

Step 3: Check Git Status
    git status

    Expected output:
    - Shows "On branch main"
    - Lists any uncommitted files

Step 4: Stage All Files
    git add -A

    This adds all new and modified files

Step 5: Commit Files
    git commit -m "Add complete Flask web app with deployment files and documentation"

    Expected output:
    - Shows number of files changed
    - Shows insertions/deletions

Step 6: Push to GitHub
    git push origin main

    Expected output:
    - Shows "Everything up-to-date" or
    - Shows files being pushed

Step 7: Verify Success
    git log --oneline -n 3

    Should show your commits

================================================================================
                        IF PUSH FAILS
================================================================================

Error: "fatal: not a git repository"
────────────────────────────────────────────────────────────────────────────────
Solution:
    You're not in the right directory
    Make sure: cd C:\Users\abhis\TT_Project


Error: "Authentication failed"
────────────────────────────────────────────────────────────────────────────────
Solution:
    Use GitHub token instead of password
    1. Generate token: https://github.com/settings/tokens
    2. Use token as password when prompted
    Or use GitHub Desktop (easier)


Error: "Permission denied"
────────────────────────────────────────────────────────────────────────────────
Solution:
    Check that you're the repository owner
    Go to: https://github.com/settings/repositories


Error: "Nothing to commit"
────────────────────────────────────────────────────────────────────────────────
Solution:
    All files already committed
    Just verify on GitHub.com


Error: "Your branch is ahead of origin"
────────────────────────────────────────────────────────────────────────────────
Solution:
    Just push the commits:
    git push origin main
