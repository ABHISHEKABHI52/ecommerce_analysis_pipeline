================================================================================
                          TROUBLESHOOTING
================================================================================

PROBLEM: "Port 5000 already in use"
SOLUTION: Use a different port or kill process using it
    python app.py --port 5001

PROBLEM: "Module not found" errors
SOLUTION: Install dependencies
    pip install -r requirements.txt

PROBLEM: "Templates not found"
SOLUTION: Ensure templates/ folder is in same directory as app.py
    Correct: C:\Users\abhis\TT_Project\templates\
    Wrong: C:\Users\abhis\TT_Project\app\templates\

PROBLEM: Charts not showing in web app
SOLUTION: Charts use base64 encoding (embedded in HTML)
    They should show automatically - no action needed

PROBLEM: Deployment failing on Render/Heroku
SOLUTION: Check logs for specific errors
    Render: View logs in dashboard
    Heroku: Run: heroku logs --tail

PROBLEM: "No module named 'gunicorn'"
SOLUTION: For cloud deployment only - already in requirements.txt
    Local testing uses: python app.py
    Cloud deployment auto-installs via requirements.txt
