STEP 1: PUSH TO GITHUB (If not already done)
================================================================================

If your project is on GitHub, skip to STEP 2.

Otherwise, run these commands in your terminal:

    git remote add origin https://github.com/YOUR_USERNAME/TT_Project.git
    git branch -M main
    git push -u origin main

Then refresh your GitHub page to verify all files are uploaded.

Time needed: 5 minutes
Status: Critical for Render deployment


STEP 2: CREATE RENDER ACCOUNT & CONNECT GITHUB
================================================================================

1. Go to https://render.com
2. Click "Sign Up"
3. Choose "Sign up with GitHub" (recommended)
4. Authorize Render to access your GitHub repositories
5. Complete the setup

Time needed: 5 minutes
Status: Quick & easy


STEP 3: DEPLOY YOUR APP
================================================================================

1. Go to Render Dashboard: https://dashboard.render.com
2. Click "New +" → "Web Service"
3. Click "Connect GitHub"
4. Select your "TT_Project" repository
5. Fill in these settings:

   Name:            ecommerce-analysis
   Region:          Oregon (US)
   Branch:          main
   Build Command:   pip install -r requirements.txt
   Start Command:   gunicorn app:app
   Plan:            Free

6. Click "Create Web Service"
7. Wait 2-3 minutes for deployment
8. Your app will be live at:
   https://ecommerce-analysis-xxxx.onrender.com

Time needed: 10-15 minutes (including build time)
Status: Your app goes LIVE after this step!
