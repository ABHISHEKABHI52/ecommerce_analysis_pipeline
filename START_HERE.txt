================================================================================
                        STEP-BY-STEP: RENDER DEPLOYMENT
================================================================================

This is the fastest and easiest path to live app.

STEP 1: ENSURE PROJECT IS ON GITHUB
────────────────────────────────────────────────────────────────────────────────

If your project is NOT on GitHub yet:

    a) Create account: https://github.com
    b) Create new repository: TT_Project
    c) In command prompt, run:

       cd C:\Users\abhis\TT_Project
       git remote add origin https://github.com/YOUR_USERNAME/TT_Project.git
       git branch -M main
       git push -u origin main

    d) Verify: Go to GitHub and confirm all files are there

If already on GitHub: Skip this step

Time: 5-10 minutes


STEP 2: CREATE RENDER ACCOUNT
────────────────────────────────────────────────────────────────────────────────

    a) Go to: https://render.com
    b) Click "Sign Up"
    c) Select "Sign up with GitHub" (recommended)
    d) Authorize Render to access GitHub
    e) Complete email verification

Time: 5 minutes


STEP 3: DEPLOY YOUR APP
────────────────────────────────────────────────────────────────────────────────

    a) Go to: https://dashboard.render.com
    b) Click "New +" button
    c) Click "Web Service"
    d) Click "Connect GitHub"
    e) Find and select: TT_Project
    f) Fill in these fields:

       Name:          ecommerce-analysis
       Region:        Oregon (US)
       Branch:        main
       Build:         pip install -r requirements.txt
       Start:         gunicorn app:app
       Plan:          Free

    g) Click "Create Web Service"
    h) Wait for "Deploy successful" (2-3 minutes)

Time: 10-15 minutes


STEP 4: TEST YOUR APP
────────────────────────────────────────────────────────────────────────────────

    a) Get your app URL from Render dashboard
       (Looks like: https://ecommerce-analysis-xxxx.onrender.com)

    b) Visit your app URL
    c) Test all pages:
       - Dashboard (should show metrics)
       - Data (should show 20 products)
       - Visualizations (should show 4 charts)
       - Analysis (should show insights)

    d) Test API: Visit /api/stats
       Should return JSON data

Time: 5-10 minutes

TOTAL TIME: 25-40 minutes

================================================================================
                        TROUBLESHOOTING
================================================================================

Problem: "Port 5000 already in use" (Local only)
Solution: Use different port: python app.py --port 5001

Problem: "Module not found" errors
Solution: Install dependencies: pip install -r requirements.txt

Problem: Charts not showing
Solution: This is normal. Charts are embedded in HTML automatically.

Problem: Build failed on Render
Solution: Check Render logs for specific error message

Problem: App crashes after deploying
Solution: Check Render logs. Likely missing file or config issue.

Problem: Can't connect GitHub to Render
Solution: Re-authorize in Render settings

For more help, see: RENDER_DEPLOYMENT.md
