STEP 1: ENSURE PROJECT IS ON GITHUB

  d) Verify on GitHub: Your repository should show all project files

Time: 5-10 minutes
Critical: Required for Render deployment


STEP 2: CREATE RENDER ACCOUNT
────────────────────────────────────────────────────────────────────────────────

  a) Go to https://render.com
  b) Click "Sign Up"
  c) Select "Sign up with GitHub" (recommended)
  d) Authorize Render to access your GitHub
  e) Complete setup

Time: 5 minutes
Benefit: Automatic GitHub integration


STEP 3: DEPLOY YOUR APP
────────────────────────────────────────────────────────────────────────────────

  a) Go to Render Dashboard: https://dashboard.render.com
  b) Click "New +" → "Web Service"
  c) Click "Connect GitHub"
  d) Select "TT_Project" repository
  e) Fill in configuration:

     Field Name           Value
     ────────────────────────────────────
     Name                 ecommerce-analysis
     Region              Oregon (US)
     Branch              main
     Root Directory      (leave blank)
     Build Command       pip install -r requirements.txt
     Start Command       gunicorn app:app
     Plan                Free

  f) Click "Create Web Service"
  g) Wait 2-3 minutes for build completion
  h) Your app will be live at:
     https://ecommerce-analysis-xxxx.onrender.com

Time: ~15-20 minutes (including build)
Result: Your app is LIVE on the internet!
